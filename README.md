# LinkedList in typescript
Simple linked-list implementation in typescript 

Available methods:
```javascript
insert(data: T): void;

insertAt(data: T, index: number): void;

insertAtLast(data: T): void;

remove(index: number): void;

get(index: number): T;

toArray(): T[];

toArrayReversed(): T[];

length(): number;

clear(): void;

forEach(): void;

consoleLog(): void;
```
